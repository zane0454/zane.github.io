Markdown 适合用来维护长期内容。它接近纯文本，容易编辑，也不依赖特定软件。

## 标题

```markdown
# 一级标题
## 二级标题
### 三级标题
```

## 链接与图片

```markdown
[链接文字](https://example.com)
![图片说明](./image.jpg)
```

## 代码块

使用三个反引号包裹代码，并在开头指定语言，即可获得语法高亮。

## 提示

一篇文章最好只有一个一级标题。正文从二级标题开始，可以让目录结构更清楚。
